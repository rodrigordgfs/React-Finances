
# React Finances

WebApp para cadastrar receitas e despesas mensais.


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/rodrigordgfs/React-Finances
```

Entre no diretório do projeto

```bash
  cd react-finances
```

Instale as dependências

```bash
  yarn add 
  ou
  npm i
```

Inicie o servidor

```bash
  yarn dev
  ou
  npm run dev
```
